# PasswordValidation

Sample solution providing a simple password validator with tests.

## Pre-requisites

- git
- dotnet core

## Setup

Clone the repository

```
git clone git@github.com:stormwild/PasswordValidation.git
cd PasswordValidation
```

Run the tests

```
dotnet test
```
